# Wishpool Buddy - 「三角色体系」产品需求文档 (PRD)

## 1. 产品概述
Wishpool Buddy 致力于提供情绪化、私密化的“心愿搭子”体验。本次迭代核心引入「三角色体系」（眠眠月、芽芽星、软软云）。

**核心设计理念 (Design Thinking)** 
遵循前端高质量设计原则，拒绝千篇一律的模板UI。系统从单一的暗黑模式，升级为**“一搭子一宇宙”**：用户切换角色时，界面的字体(Typography)、色彩(Themes)、动效(Motion)及呼吸感(Composition)都会发生深度的沉浸式变化。

---

## 2. 角色前端视觉规范 (Aesthetic Directions)

### 2.1 🌙 眠眠月 (Mianmian Yue) —— 【深夜水墨·月光容器】
* **定位**：治愈系 / 睡前放松、冥想
* **视觉材质**：珠光树脂 (Pearl Resin)，立体圆润，边缘散发鹅黄光晕
* **前端美学**：Refined Minimalism (极致克制与优雅)
  * **Color Theme**: 深靛蓝黑 `#0a0e1a` 为主，月光金 `#f5c842` 做高亮。
  * **Typography**: 古典温度的衬线体 (如 **Noto Serif SC** 直出) 营造“杂志级排版”。
  * **Motion**: 暗夜星空背景 (`StarField`闪烁)。涟漪交互产生径向发散动画。
  * **漂流瓶意象**：轻柔诗意的心愿（发呆、听雨）。

### 2.2 🌱 芽芽星 (Yaya Xing) —— 【深空极光·荧光果冻】
* **定位**：成长系 / 学习充电、探索
* **视觉材质**：透明果冻 (Clear Jelly)，内部折射微光，头顶生机勃勃的绿芽
* **前端美学**：Retro-Futuristic (探索与好奇心)
  * **Color Theme**: 太空深紫 `#1a0f2e` 底色，霓虹薄荷绿 `#4ade80` 穿插。
  * **Typography**: 具有圆润转角或未来主义的粗细对比字体，增加探索感。
  * **Motion**: “极光流转”背景 (Gradient Mesh 透明流线渐变)。按钮出现“Q弹悬浮回弹”。
  * **漂流瓶意象**：探索类高动机心愿（新技能、未知街道）。

### 2.3 ☁️ 软软云 (Ruanruan Yun) —— 【晨风白昼·植绒呼吸】
* **定位**：梦幻系 / 心情记录、发呆
* **视觉材质**：植绒质感 (Flocked Velvet)，极致柔软，散发柔和白/桃粉发光
* **前端美学**：Soft Pastel / Airy (彻底摒弃黑夜模式的白昼疗愈风)
  * **Color Theme**: 晨曦蓝 `#e0f2fe`、治愈白 `#ffffff` 与浅桃粉 `#ffedd5`。卡片为磨砂毛玻璃 (`backdrop-filter`)。
  * **Typography**: 圆滑、无锋利的友好型字体 (如 **Nunito** 或 **Fraunces**)。
  * **Motion**: 节奏极其舒缓的“云朵呼吸上下浮动”背景。UI 互动带来像“云雾扩散”一样柔和的涟漪。
  * **漂流瓶意象**：松弛感心愿（放空、吹风）。

## 3. 核心交互链路落地 (Key Flows)

### 3.1 角色首次登场与无缝切换
* **交互路径**：页面左上角的 **Logo/头像区域** 作为全局角色切换入口。点击Logo区弹出或下拉展示“陪伴羁绊”打卡列表，供用户直接点击切换。
* **高光动效 (Motion Priorities)**：采用 CSS 层面的平滑渐隐过度（非生硬刷页）。当选出“软软云”并设为搭子后，深渊星空要通过类似于破晓的 `transition` （延迟控制及多时间轴动画），缓缓变成温暖毛茸茸的白夜状态。

### 3.2 意图识别动画 (Intent Processing)
思考动画针对各材质必须重新实现：
* **眠眠月**：沿中心轻缓自转。
* **芽芽星**：头顶枝芽微张，周围环绕绿色数字/知识粒子。
* **软软云**：大幅度的懒散舒展呼吸，边界毛边产生轻微的微风拂动感。
