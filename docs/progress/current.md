## 最近更新（2026-04-07）

### 交互细节优化
1. **首页长按流程** - 长按荧光圈后进入聊天页并自动打开语音输入
2. **聊天页长按崩溃修复** - 改用事件机制触发语音输入，避免状态冲突
3. **语音输入 UI 重构** - 改为 `fixed` 定位，全屏遮罩完全覆盖导航栏和输入框
4. **光晕效果增强** - 为荧光圈和荧光条添加多层阴影光晕效果
5. **设置栏定位修复** - z-index 提升到 50，确保正确显示
6. **气泡定位修复** - 改为居中对齐
