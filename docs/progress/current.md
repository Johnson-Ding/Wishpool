## 最近更新（2026-04-08）

1. **首页长按流程收口**：长按荧光圈时先打开语音，完成后再进入聊天页并插入对应卡片。
2. **语音输入挂载上移**：`VoiceInputOverlay` 提至 `MainTabScreen` 层，避免聊天页内状态冲突。
3. **浮层坐标系统一**：设置面板、语音遮罩、气泡统一挂到 `#phone-demo-overlays`，不再使用全局 `fixed` 口径。
4. **聊天输入补全发送动作**：支持回车 / 点击发送，避免 mock 提交卡死。
5. **设置口径收口**：当前保留角色选择与视觉联动，不把“独立主题切换”继续当作单独功能描述。
