# PRD — 角色与设置（V4）

> **版本**：V4.0  
> **状态**：当前基准
> **说明**：文件名沿用历史命名，当前 V4 的实际重点是角色选择、设置面板与视觉联动，不再把“独立主题切换”作为单独主流程功能

---

## 目标

这个板块在 V4 中负责三件事：
- 给用户一个稳定、低打扰的设置入口
- 让用户选择当前最契合自己的陪伴角色
- 让角色头像、对话气质与视觉 token 保持一致联动

---

## 当前范围

- 设置入口：我的愿望 Tab 右上角设置 icon
- 设置面板：底部升起的面板
- 角色选择：眠眠月 / 朵朵云 / 芽芽星
- 视觉联动：角色头像、页面视觉 token、荧光圈 / 荧光条氛围联动
- 基础设置项：更新、反馈、会员状态等当前 demo 占位内容

---

## 不在当前范围

- 不做独立的主题商城、多套皮肤系统
- 不把“主题颜色切换”作为当前 demo 的单独验收项
- 不要求接真实后端保存设置
- 通知、隐私、账号等设置本轮只保留结构占位，不展开真实闭环

---

## 三个角色

| 角色 | 情绪定位 | 对话作用 | 当前可见联动 |
|---|---|---|---|
| 眠眠月 | 温柔、安静 | 接住情绪、降低表达门槛 | 头像、文案气质、视觉 token |
| 朵朵云 | 轻盈、明快 | 行动推进、把愿望拆成动作 | 头像、文案气质、视觉 token |
| 芽芽星 | 活力、希望 | 灵感回流、从碎碎念里发现潜在愿望 | 头像、文案气质、视觉 token |

---

## 设置面板结构

### 角色选择区
- 当前角色头像（居中，大尺寸）
- 三个角色选项（横排，小头像 + 名称）
- 点击后立即切换，无需确认
- 切换后更新当前角色相关的视觉表现与头像显示

### 其他设置项
- 会员状态
- 问题反馈
- 更新检查

---

## 当前 demo 规则

- 角色选择使用本地状态
- 当前会把所选角色持久化到 `wishpool_character`
- 允许角色视觉联动，但不再额外拆出“主题切换入口”来定义
- 重点验证：设置面板交互、角色切换反馈、视觉一致性

---

## 验收口径

- 用户能从“我的”进入设置面板
- 用户能明确看到并切换当前角色
- 切换后角色头像与视觉联动即时生效
- 刷新后仍保留最近一次角色选择
